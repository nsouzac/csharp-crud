# CRUD Project with Blazor and MySQL Integration

This is a simple, yet functional, CRUD (Create, Read, Update, Delete) project designed to work with a front-end interface using Blazor and integrated with a MySQL database.

## Features

- **Create**: Add new records to the database.
- **Read**: Retrieve and display records from the database.
- **Update**: Modify existing records in the database.
- **Delete**: Remove records from the database.

## Tech Stack

- **Front-End**: Blazor (WebAssembly)
- **Back-End**: C# (ASP.NET Core)
- **Database**: MySQL
- **ORM**: Entity Framework Core

## Prerequisites

- .NET 6.0 or higher
- MySQL server installed and running
- Visual Studio or any C# compatible IDE
